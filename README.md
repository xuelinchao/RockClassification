# 岩识（原创名称）
（如引用请注明出处及作者:中国地质大学-信息工程学院-薛林超）

利用深度学习模型对岩石图像进行分类
初步实现两个主要功能
1. 从系统相册选择图片识别
2. 自定义相机实时识别

下一步想要实现
1. 对选择的图片可以进行手动随意选择任意大小的区域进行识别
2. 对相机的实时识别模块加入取景框，取景框可以通过手指进行放大缩小，并对取景框内的图像进行识别
